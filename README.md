# Photorama — Galerie d'images intelligente

Application Java qui détecte automatiquement les photos dupliquées dans un dossier et les regroupe visuellement. Trois algorithmes de comparaison d'images sont implémentés, accessibles via une interface graphique JavaFX ou en ligne de commande.

---

## Fonctionnalités

- Détection de doublons avec trois algorithmes différents
- Comparaison pixel par pixel (Algo 1)
- Hachage basé sur la moyenne de l'image (Algo 2)
- Hachage basé sur les différences de voisins (Algo 3)
- Interface graphique JavaFX avec galerie interactive
- Tolérance aux différences ajustable (faible ou élevée)
- Rapport en ligne de commande avec informations de débogage
- Gestion des images corrompues sans plantage

# Photorama — Smart Image Gallery

Java application that automatically detects duplicate photos in a folder and groups them visually. Three image comparison algorithms are implemented, accessible through a JavaFX graphical interface or via the command line.

---

## Features

- Duplicate detection with three different algorithms
- Pixel-by-pixel comparison (Algorithm 1)
- Average-based image hashing (Algorithm 2)
- Neighbour difference-based hashing (Algorithm 3)
- JavaFX graphical interface with interactive gallery
- Adjustable difference tolerance (low or high)
- Command-line report with debug information
- Corrupted image handling without crashing
